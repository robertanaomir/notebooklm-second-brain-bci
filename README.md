# 🧠 NotebookLM Segundo Cérebro – Interfaces Cérebro-Computador (BCI)

Projeto desenvolvido como desafio da DIO utilizando o Google NotebookLM para construir um **Segundo Cérebro** sobre Interfaces Cérebro-Computador (Brain-Computer Interfaces – BCI).

O projeto demonstra como utilizar Inteligência Artificial como ferramenta de aprendizagem ativa, reunindo fontes confiáveis, realizando engenharia de prompts e organizando o conhecimento em um guia de estudo reutilizável.

---

# Objetivos

- Estudar os fundamentos das Interfaces Cérebro-Computador (BCI).
- Organizar o conhecimento utilizando o Google NotebookLM.
- Aplicar técnicas de Engenharia de Prompts.
- Consolidar o aprendizado em um guia estruturado.
- Documentar todo o processo em um repositório GitHub.
