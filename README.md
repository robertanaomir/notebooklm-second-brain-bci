NotebookLM Second Brain – Brain-Computer Interfaces (BCI)

Projeto desenvolvido como desafio da DIO utilizando o Google NotebookLM para construção de um Segundo Cérebro sobre Interfaces Cérebro-Computador (Brain-Computer Interfaces - BCI).

O objetivo foi utilizar Inteligência Artificial como ferramenta de aprendizagem ativa, organizando informações provenientes de diferentes fontes, realizando engenharia de prompts e consolidando o conhecimento em um guia didático para pessoas sem conhecimento prévio sobre o tema.
