# Fontes Utilizadas

As seguintes fontes foram utilizadas como base para construção do caderno temático:

- Artigos sobre Interfaces Cérebro-Computador (BCI)
- Publicações acadêmicas sobre EEG e processamento de sinais neurais
- Conteúdos introdutórios sobre Neurociência
- Materiais utilizados no Google NotebookLM

Todas as respostas geradas foram comparadas com as fontes disponibilizadas no NotebookLM.
